# fresh-cart platform

### Installation Steps

#### 2. Install Dependencies:

```
npm install

```

#### 3. Create a .env file:

```
cp .env.example .env

```

### Run The Service

#### 1. Start the development server:

```
npm start

```

### Testing with Jest

This project uses Jest for unit and integration testing of React components with TypeScript.

#### To run all tests once:

```
npm test

```

#### To run tests in watch mode:

```
npm run test:watch

```
